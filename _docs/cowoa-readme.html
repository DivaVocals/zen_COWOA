PURPOSE

  • COWOA 2.6 lets your customer checkout without having to register an account on your store.
  • Please check your host's PHP version. This module will NOT work for hosting configurations using PHP versions older than 5.2.x
  • COWOA 2.6 was tested for Zen Cart 1.5.0, Zen Cart 1.5.1 Zen Cart 1.5.3 compatibility only.
    Any issues which arise when attempting to use this module with Zen Cart 1.3.9 may be attributed to the differences between the older Zen Cart versions.
  • If you are still running an older version of Zen Cart we strongly recommend that you upgrade as there have been many security enhancements made.
  • COWOA 2.6 IS NOT COMPATIBLE WITH VERSIONS OF ZEN CART PRIOR TO ZEN CART 1.5.0!!
  • There will be no support provided for Zen Cart 1.3.9 with 2.6 of COWOA.

New Installation For Zen Cart 1.5, 1.5.1 and 1.5.3 Brand New Installation Of COWOA:

  1. Download and Unzip the package contents to a temp directory
  2. Change and Rename the YOUR_ADMIN inside the New_Installation/1_Store_Filesfolder to match the name of your shop's admin folder name.. Let me repeat this as this is the one step in a module install that most OFTEN gets overlooked/missed:

    Change and Rename the Admin_Folder inside the New_Installation folder to match the name of your shop's admin folder name..
  3. Change and Rename all the YOUR_TEMPLATE folder names in the New_Installation/1_Store_Files/includes folders to match the name of your template's folder name

  4. Upload ALL the files in the New_Installation/1_Store_Files folder to your store directory. The directory structure within these folders is like your Zen Cart store's folder structure if you have renamed them correctly. For most FTP clients the directory structure of the /includes directory should be preserved, and the files should "fall" into their proper places. If your FTP client isn't capable of this, you might have to manually place each file according to where they appear in this module.
  5. Install the SQL Patch. To do this, open up the file COWOA_Full_Install.sql in a text editor to copy and paste the file. In your admin panel, open Tools -> Install SQL Patches. Copy the query from COWOA_Full_Install.sql into the text box and hit 'send'. This will add the necessary COWOA fields and setup COWOA inthe Configuration Section. It will also set Split Login to True which is necessary for this mod to work.

Basic Un-Install Instructions
  1. Remove and replace all the files that were over written by COWOA with your backed up files. You did make a copy Right?
  2. Run the Uninstall_COWOA.sql using the "SQL Query Executor" tool in the Admin (Tools > Install SQL Patches) to remove the database entries.

HOW TO USE

COWOA is EASY to use. Checkout without and account module looks similar to the (regular checkout) page.

To switch COWOA on

Go to ADMIN->Configuration->COWOA Configuration. There are 5 configuations that need to be set to True or False..

  • Activate COWOA Checkout.
  • Turn Off Sideboxes During Checkout
  • Enable The Order Status Function of COWOA.
  • Enable The E-Mail Order Function of COWOA.
  • Enable The Forced Logoff Function of COWOA.

That's it you now have Checkout Without an Account Installed..
